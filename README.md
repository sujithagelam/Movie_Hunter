An interactive **Quiz Application** built with **React** that tests users with multiple-choice questions, a countdown timer, and a beautiful dark/light theme.

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/sujithagelam/genquiz.git
```

### Navigate to project folder

```bash
cd genquiz
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

---

## 🎯 How to Play

1. Read the question carefully.
2. Select one option.
3. Correct answers are highlighted in green.
4. Wrong answers are highlighted in red.
5. Click **Next** to continue.
6. If time runs out, the quiz automatically moves to the next question.
7. View your final score at the end.
